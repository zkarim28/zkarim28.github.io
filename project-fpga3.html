Drum Synthesizer
2D Wave Equation in Hardware
FPGA Design Project

As part of Cornell’s Hardware Acceleration via FPGA course (ECE 5760), I designed and implemented a drum synthesizer on an FPGA. The project involved creating a hardware implementation of the 2D wave equation to simulate the behavior of a drumhead. By leveraging the parallel processing capabilities of the FPGA, I was able to achieve real-time audio synthesis, allowing for dynamic and responsive drum sounds. This project not only deepened my understanding of digital signal processing and hardware design but also showcased the potential of FPGAs in creative applications such as music synthesis.
