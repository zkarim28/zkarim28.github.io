Summary

This project involves creating a GUI application using Projucer to develop a live MIDI input visualizer. The visualizer dynamically changes the intensity of note colors based on velocity. Additionally, a new feature displays a keyboard at the bottom of the screen, highlighting each key in blue when played. This enhances the visual representation of live MIDI input, making it more interactive and informative.
